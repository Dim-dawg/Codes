# Sheet Specifications: Budget Entity View

This document defines the layout and formatting for the monthly budget sheets generated by `sync_entity_budget.py`.

## 1. Sheet Layout

| Range | Content | Format |
|-------|---------|--------|
| **A1** | "Year:" | Bold, Left, Background: #F2F2F2 |
| **B1** | **[Year Value]** | **Number (0)**, Bold, Left, Background: #F2F2F2 |
| **A2** | "Month:" | Bold, Left, Background: #F2F2F2 |
| **B2** | **[Month Name]** | **Text (@)**, Bold, Left, Background: #F2F2F2 |
| **C1** | "Status:" | Bold, Right, Background: #F2F2F2 |
| **D1** | **[Current Status]** | Text (e.g., "✅ Ready", "🔄 Syncing...") |
| **C2** | "Sync Now:" | Bold, Right, Background: #F2F2F2 |
| **D2** | **[Checkbox]** | **Checkbox** (TRUE/FALSE) |
| **A3:AH3**| Table Headers | Background: #444444, Text: White, Bold, Center |
| **A4:A** | Profile Names | Left, Background: Alternating White/LightGrey |
| **B4:B** | Planned Amount| Currency ($#,##0.00), Center |
| **C4:C** | Actual Amount | Currency ($#,##0.00), Center |
| **D4:AH** | Daily Actuals | Currency ($#,##0.00), Center |

## 2. Formatting Rules

- **Frozen Panes:** Top 3 rows and Column A are frozen.
- **Number Formats:**
  - Year (B1): `0` (Standard Number, no decimals, no commas).
  - Month (B2): `@` (Plain Text).
  - Amounts (B4:AH): `$#,##0.00;$(#,##0.00)` (Currency).
- **Colors:**
  - Section Headers:
    - INCOME: #1D9E75 (Green)
    - EXPENSE: #D85A30 (Red)
    - SAVINGS: #185FA5 (Blue)
  - Section Totals: Same as Section Header background, White text.
- **Column Widths:**
  - A: 220px
  - B-C: 110px
  - D-AH: 42px

## 3. Data Source
- **Profiles:** Fetched from `public.profiles`.
- **Budgets:** Fetched from `public.budgets` for the specific `month` string (YYYY-MM).
- **Transactions:** Fetched from `public.transactions` filtered by date range.
